| **Criterion**                       | **Meaning / Purpose**                                                  | **Typical Metrics**        |
| ----------------------------------- | ---------------------------------------------------------------------- | -------------------------- |
| **1. Segmentation Accuracy**        | Đánh giá độ chính xác trong việc xác định vùng bệnh và biên đối tượng. | mIoU, Dice, F1-score       |
| **2. Multi-class Capability**       | Khả năng phân biệt nhiều loại bệnh trên nhiều loại cây khác nhau.      | Confusion Matrix, Macro F1 |
| **3. Processing Speed**             | Tốc độ xử lý ảnh hoặc video trong quá trình suy luận.                  | FPS, Latency (ms/frame)    |
| **4. Robustness Across Conditions** | Giữ hiệu năng ổn định dưới thay đổi ánh sáng, nền, góc chụp.           | Cross-condition testing    |
| **Criterion**                             | **Meaning / Purpose**                                                 | **Typical Metrics**                                    |
| ----------------------------------------- | --------------------------------------------------------------------- | ------------------------------------------------------ |
| **5. Deployability**                      | Khả năng chạy trên thiết bị thực như mobile, drone, hoặc edge device. | Model size, CPU/GPU speed, TFLite/ONNX support         |
| **6. Data Efficiency**                    | Mức độ phụ thuộc vào dữ liệu gán nhãn khi huấn luyện.                 | Number of required samples, augmentation effectiveness |
| **7. Ease of Use**                        | Dễ tích hợp vào hệ thống giám sát hoặc ứng dụng thực tế.              | API support, UI/UX testing                             |
| **8. Generalization & Field Performance** | Khả năng hoạt động tốt trên dữ liệu thực địa ngoài phòng lab.         | Cross-dataset test, real-field dataset evaluation      |
